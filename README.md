# CI/CD Automated Website Deployment using Jenkins, Docker, GitHub, and AWS EC2 on Netflix Clone

## Project Overview

This project demonstrates the implementation of a CI/CD pipeline using Jenkins, Docker, GitHub, and AWS EC2 for automated website deployment.

## Technologies Used

- AWS EC2
- Jenkins
- Docker
- GitHub
- HTML
- CSS
- JavaScript
- Linux

## Workflow

GitHub → Jenkins → Docker → AWS EC2 → Live Website

## Features

- Automated Deployment
- Docker Containerization
- GitHub Integration
- Jenkins CI/CD Pipeline
- AWS Cloud Hosting
